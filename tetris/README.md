# Happy Tetris

Яркий тетрис для Android, написанный на Java без внешних библиотек.

В игре есть анимированная стартовая заставка, сохранение рекорда, комбо,
частицы, звуки, вибрация, ghost-фигура и плавное управление жестами.

## Управление

- Касание игрового поля: поворот фигуры.
- Свайп влево или вправо: перемещение.
- Свайп вниз: ускоренное падение.
- Нижние кнопки: влево, поворот, вправо, вниз и мгновенный сброс.
- Кнопка в правом верхнем углу: пауза.

## Сборка

```powershell
.\gradlew.bat assembleDebug
```

APK создаётся в `app\build\outputs\apk\debug\app-debug.apk`.
