# Ozon Witch

Мультяшная 2D-игра для Android на Java.

Ведьма работает у пункта выдачи и склада, ходит по нижней дорожке,
собирает обычные посылки и стреляет магией в клиентов с криком «Возврат!».
Попадание превращает клиента в жабу.

## Новая механика

- Персонаж использует анимации из набора PIPOYA FREE 2D Game Character Sprites.
- Ведьма ходит по земле и умеет прыгать через коробки, конусы и тележки.
- Клиенты появляются только на нижней дорожке.
- Сверху падают товары-потеряшки. Столкновение снимает 20 очков.
- За каждые 100 очков ведьма проигрывает анимацию радости.
- Столкновение с клиентом или препятствием отнимает одну из трёх жизней.

## Управление

- Левая половина: веди пальцем влево или вправо для перемещения.
- Нажми в верхней части левой половины, чтобы прыгнуть.
- Правая половина: нажми или удерживай для магической стрельбы.

## Сборка

```powershell
.\gradlew.bat assembleDebug
```

APK: `app\build\outputs\apk\debug\app-debug.apk`.

## Графика

Спрайты игрового персонажа взяты из предоставленного пользователем архива
`PIPOYA FREE 2D Game Character Sprites.zip`.
